  DATABASE COLLECTIONS & METHODS

DB.articles (Article Management)
├── get()                      - Get all articles
├── add(article)              - Create new article
├── delete(id)                - Delete by ID
├── update(id, data)          - Update article
├── findById(id)              - Find specific article
├── search(keyword)           - Full-text search
├── filterByCategory(cat)     - Filter by category
├── getFeatured(count)        - Get first N articles
└── getDefault()              - Get sample articles

DB.blogs (Blog Management)
├── get()                      - Get all blogs
├── add(blog)                 - Create new blog
├── delete(id)                - Delete by ID
├── update(id, data)          - Update blog
├── findById(id)              - Find specific blog
└── getDefault()              - Get sample blogs

DB.categories (Category Management)
├── get()                      - Get all categories
├── add(category)             - Create new category
├── delete(id)                - Delete by ID
└── getDefault()              - Get 7 default categories

DB.users (User Management)
├── get()                      - Get all users
├── add(user)                 - Create new user
├── delete(id)                - Delete by ID
├── findByUsername(name)      - Find by username
└── getDefault()              - Get default admin user

Utility Functions:
├── init()                    - Initialize all collections
├── export()                  - Export all data as JSON
├── import(data)              - Import data from JSON
├── getStats()                - Get database statistics
├── clearAll()                - Delete all data

  SUPPORT & TROUBLESHOOTING

Common Issues & Solutions:

Q: "Data not saving"
A: Check browser console (F12) for errors
   Ensure localStorage is enabled
   Check browser storage limit

Q: "Articles not showing on articles.html"
A: Refresh page (F5)
   Check browser console for errors
   Verify data exists: DB.articles.get()

Q: "Admin can't login"
A: Use exact credentials: SEMEELKT / ADMIN
   Check caps lock
   Clear browser cache

Q: "How do I backup?"
A: Open console (F12)
   Type: const backup = DB.export()
   Save to file

Q: "How do I restore?"
A: Use: DB.import(backupData)
